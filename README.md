# 🎨 主要功能與亮點

## 🎯 1. 多頁式導覽（Single Page Navigation）
使用 JavaScript 控制 `.section` 的顯示與 `.nav` 切換  

功能包含：
- 點擊首頁、About、Project、Contact 自動切換頁面  
- 手機版側邊欄會自動收合  
- 自動同步選單 active 樣式  

---

## 🌗 2. Dark / Light Mode 暗黑模式切換
使用 CSS + JS 實作主題切換  

- 可一鍵切換「深色 / 淺色」  
- 自動記住使用者偏好  
- Icon 動態切換 🌙 / ☀️  

---

## 📝 4. 動態打字效果（Typing Animation）
首頁自動輪播：

- 全端工程師  
- 軟體工程師  
- 網頁設計工程師  

使用 Typed.js + JS 控制  

---

## 🖼 5. 作品展示 Modal（Project Modal Viewer）
你的 Projects 區域可以：

- 點擊按鈕開啟 Modal  
- 依序瀏覽圖片 / 影片  
- 有 Next / Back  
- 有每一步驟的描述  
- 自動播放影片
